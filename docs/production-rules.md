# Reglas De Produccion

## Export

- Preferir PNG con transparencia.
- Mantener una version `*_chroma.png` cuando venga de generacion con fondo verde.
- No mezclar animaciones en filas sin documentar.
- Cada spritesheet debe tener un manifiesto con tamano, filas, columnas y lista de animaciones.

## Nombres

Usar `kebab-case` para carpetas y `snake_case` para archivos de assets.

Ejemplo:

`art/characters/astra-mycelion/spritesheets/astra_mycelion_movement_sheet.png`

## Revision

Antes de aceptar un recurso:

- La silueta debe ser reconocible en negro.
- La pose debe comunicar la accion sin texto.
- El personaje no debe parecerse a personajes existentes.
- El fondo debe ser transparente.
